07 — Focus Styles

Tab through each column to feel the difference. Then try clicking first, then Tab — notice how :focus-visible behaves differently.

How to test: Press Tab to navigate. In column C, also try clicking a button with the mouse first and then pressing Tab — the outline only appears when the keyboard is involved.
A — outline: none (bad)

Focus is invisible. Keyboard users have no idea where they are on the page.

Learn more 
/* The most common mistake */
* { outline: none; }
:focus { outline: none; }
B — :focus always visible

Works for keyboard — but the orange ring also appears when clicking with a mouse, which designers often dislike.

Learn more 
:focus {
  outline: 3px solid orange;
  outline-offset: 3px;
}
C — :focus-visible (correct)

Visible for keyboard users. Hidden when using a mouse. Best of both worlds.

Learn more 
:focus:not(:focus-visible) {
  outline: none;
}
:focus-visible {
  outline: 3px solid #00b894;
  outline-offset: 3px;
  border-radius: 4px;
}

Focus ring contrast

The focus ring itself must meet a contrast ratio of 3 : 1 against its surrounding colours (WCAG 2.4.11, Level AA). A light grey outline on a white background will fail. Test the ring colour with the same tools used for text contrast.

Tip: Use outline-offset to add a small gap between the element and the focus ring. This improves visibility on coloured backgrounds and avoids the ring merging with the element border.
