Press Tab to navigate. Column C: click a button with the mouse first, then press Tab — notice when the outline appears and when it does not.
A — outline: none (bad)

Focus is invisible. Keyboard users have no idea where they are.

Learn more 
* { outline: none; }
:focus { outline: none; }
B — :focus always visible

Works for keyboard users — but the ring also appears on mouse click.

Learn more 
:focus {
  outline: 3px solid orange;
  outline-offset: 3px;
}
C — :focus-visible (correct)

Visible for keyboard, hidden for mouse. Best of both worlds.

Learn more 
:focus:not(:focus-visible) {
  outline: none;
}
:focus-visible {
  outline: 3px solid #00b894;
  outline-offset: 3px;
  border-radius: 4px;
}

Focus ring contrast

The focus ring itself must meet a contrast ratio of 3 : 1 against its surrounding colours (WCAG 2.4.11). A light grey ring on a white background will fail. Test the ring colour with the same tools used for text contrast.
