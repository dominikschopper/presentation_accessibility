Why inert and not aria-hidden?

aria-hidden="true"
  • Hides from accessibility tree ✓
  • Tab focus still reaches it ✗
  • Mouse clicks still work ✗
inert
  • Hides from accessibility tree ✓
  • Blocks all Tab focus ✓
  • Blocks mouse clicks ✓

Browser support: Chrome 102+, Firefox 112+, Safari 15.5+. Polyfill: wicg-inert.
